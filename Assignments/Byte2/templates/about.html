Accountability: About This Data Set

This dataset documents the features of more than 10k different apps available on google store. Each row contains a bunch of features, and each column corresponds to a feature specific feature, such as category, rating, install times, etc. We can utilize these data to evaluate each app, therefore offering a list of recommended apps for users that are new to Android market.

Where did this data come from?

The data is downloaded from Kaggle Dataset, which can be downloaded here. As is indicated by the uploader, this data is scraped from Google Play Store.

Who has access to it?

As for the structured data used in this assignment, it is public therefore available to anyone who know the link. On the other hand, since the raw data can be accessed through a public platform, it is still available for anyone to access the raw data.
